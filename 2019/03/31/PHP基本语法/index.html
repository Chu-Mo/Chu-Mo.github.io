PHP基本语法

发表于 2019-03-31 | 分类于 自学笔记

第二章PHP基本语法
1.初识PHP脚本程序
1）语法就是PHP官方规定的一系列使用PHP的规则。
2.PHP语言标记
1）开始：<?php
结束：?>
2）<?php?>这之间就表示进入PHP模式，在开始和结束标记之外的内容都会被PHP解析器忽略。
3）可直接嵌入到HTML代码中的任意地方，在一个HTML文档中可以嵌入任意多个PHP标记，但要遵循HTML语言规则，注意文件后缀为.php文件可以在HTML代码中嵌入PHP标记，而后缀为.html的文件则不可以嵌入PHP标记。
4）文件末尾PHP代码段结束标记可以不要，在一些情况下省略掉更好
5)我们的页面最终是通过HTML，css，js来展示出一个绚丽的界面
3.指令分隔符“;”
1）PHP语句分为两种：一种是在程序中使用结构定义语句，另一种是在程序中使用功能执行语句，其中前者结尾不需要使用分号，后者必须使用分号结尾
2）结束标志?>就隐含一个分号，所以PHP代码的最后一行可以不用加“;”不加“;”时要确保PHP语句中的最后一行要有结束标记。
3）在一些PHP语句的结尾要加上“;”，表示一句话的结束，否则会报错。
4.显示乱码
一般是因为代码编码格式与项目编码格式不相符引起的，将代码全选剪切，调整好项目的编码格式粘贴即可
5.注释
/这是一个多行注释，可以有多行文字/
//这是单行注释
注释不会被PHP解析器所解析，他能帮助我们对代码进行注释方便他人也方便我们对代码的理解。
6.遇到空白的处理
空白符包括空格、tab制表符、换行，这些在PHP中都是无关紧要的（注意都是英文输入法下输入的），可以将一个语句展开成任意行，或者紧缩在一行，空格与空行的合理运用可以增强代码的清晰与可读性，如果运用不合理反而会对阅读产生负担
