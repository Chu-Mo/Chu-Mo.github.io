发表于 2019-03-31 | 分类于 自学笔记

第二章PHP基本语法——常量
1.概述：
常量是一种简单值得标识符，如其名称所暗示的，在我们PHP脚本执行期间，一个常量一旦被定义，就不能被更改或者取消定义，常量的作用域是全局，常量只能包含bool，int，float，string类型的数据！
常量是用于临时（只有在我们程序运行过程中才存在）存储值（数据）的容器。
2.定义和使用
define（常量名称,常量值）
define（’常量名称’,常量值）或者 define（”常量名称”,常量值）使用常量名称时需要去掉（’’）或者（””）,当常量的值为字符串时，字符串中的内容需要用（’字符串’或者”字符串”）的形式，否则会报错。
常量的命名与变量类似，也遵循PHP标识符的名称规则，按照惯例常量标识符总是大写的
注意：常量的命名和变量一样，只能包含字母数字下划线，并且必须以字母或者下划线开头，按照惯例常量标识符总是大写的，自己定义的常量是严格区分大小写的
define（’MY_CITY’,’上海’）
defined（）函数来检查是否定义了某个常量
3.常量和变量的区别
1）常量前没有美元符号（$）
2）常量只能用define（）函数来定义，不能通过赋值语句
3）常量可以不用理会变量范围的规则而在任何地方定义和使用
4）常量一旦被定义就不能被重新定义或者取消定义
5）常量的值只能是bool，int，float，string类型
4.预定义常量
预先定义好的常量，就是程序中可以直接使用来完成一些特殊功能

预定义常量说白了就是PHP内核已经帮我们定义好了的常量
查看预定义常量print_r(get_defined_constants());右击查看网页源码。
其中有的预定义常量是__***__形式的，这些预定义常量我们又叫他魔术常量，代码所在的位置不同，它的值也是不同的，所以他叫魔术常量。

注意：可以在www.php.net 中查看php手册。
