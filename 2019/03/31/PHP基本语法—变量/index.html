PHP基本语法—变量

发表于 2019-03-31 | 分类于 自学笔记

第二章PHP基本语法——变量
1.变量
变量是用于临时（只有在我们的程序运行过程中才会存在）存储（数据）的容器，变量在任何语言中都是核心地位。另外PHP脚本语言是一种弱类语言，和其他语言不同的是变量或者常量的数据类型由程序的上下文决定。
2.变量的声明：
PHP的特性之一就是他不要求在使用变量前声明变量，当第一次给一个变量赋值时，你才创建了变量，变量用于储值比如数字、文本字符串或者数组。一旦设置了某个变量，我们就可以在脚本中重复使用它。PHP中变量必须使用一个美元符号$后面跟着变量名来表示，使用赋值操作符（=）给一个变量赋值。
3.声明：就是事先告知，通知！
注意：PHP程序中，使用变量前不需要声明，而其他语言是需要声明的，比如Java、c语言
4.变量的释放/变量的销毁：
unset（$变量的名称）函数释放指定变量
$and1=1;
unset($and1);//销毁$and变量
echo $and1;//提示出错，因为变量已经被销毁了！
5.示例：
现实生活中我们在家里会用柜子来存放我们的物品，那么我们在编程语言中是没有物品这个概念的，只有数据，这些数据就是存放在我们的变量里的。
6.变量的命名：
1）变量名是严格区分大小写的。
$name $name=1
$NAME $NAME=1
$Name $Name=1
2)变量名由字母或者下划线开头，后面可以连接任意数量的字母，数字，或者下划线。
注意：变量名的标识符一定不要以数字开头，中间不可以使用空白字符，不能使用点分开。
总结：变量名只能包含字母、数字、下划线，并且必须以字母或者下划线开头。
3）PHP中有一些标识符是系统定义的，也称为关键字，是PHP语言的组成部分，因此不能使用他们中的任何一个作为常量函数名或类名。但是和其他语言不同的是可以在PHP中作为变量名称使用，不过这样容易混淆，所以建议还是不要使用PHP的关键字作为变量名称。
7.可变变量
一个变量的名字可以动态的设置和使用
$name=‘shanghai’；
$$name=’beijing’;
echo $name
echo $shanghai
8.变量引用赋值
引用赋值可以简单的理解为给变量起个别名。
$a=1000;
/$b=$a;//相当于把$a的值，复制一份再赋值给$b这个变量/
$b=&$a;//相当于给$a起了一个别名，操作其中任何一个，都会影响到另外一个变量的值!
$b=20;
echo $a;
9.变量类型
概述：变量类型是指保存在该变量中的数据类型，在计算机编程语言世界中每个数据都有它的类型。
具有相同类型的数据才能被彼此相互操作。
在强类型的语言中，例如C语言，变量要首先指定类型，然后才可以存储对应指定的数据，就好比现实生活中在制作一个柜子之前要决定好这个柜子是什么类型的柜子，是书柜只能存书，是衣柜只能存衣服。
但是PHP是一种弱类型的语言，和其他语言不同的是，变量或者常量的数据类型由程序的上下文决定，不需要事先声明变量的类型。一个柜子你要是存书就是书柜，要是存衣服就是衣柜，根本不需要事先声明这是书柜还是衣柜，一切根据实际需求使用即可。
10.变量类型的简介
使用var_dump（变量名）可以输出变量的类型
1）bool (布尔型)
$a=true;
var_dump($a);
往变量里面存true或者false，那么这个变量就是布尔型了，
布尔型是最简单的类型，布尔型表达了true或false，即真或假，以下值被认为是false，其他的值都被认为ture：（布尔型false，0， 浮点型0.0， 空白字符串和字符串0， 没有成员的数组，NULL注意：空白字符是指’’或者””并不是空格符）
2）int (整型)
往变量里面存整数，那么这个变量就是整型变量。
3）float (浮点型，也称double)
往变量里面存小数，那么这个变量就是浮点型数据。
$a=1.1;
var_dump($a);
4）string (字符串)
字符串就是一系列的字符组成串在一起！
5）NULL
null表示一个变量没有值，表示空
将变量直接赋值为null
声明的变量尚未被赋值
被unset函数销毁的变量
$a=1111;
unset($a);
var_dump($a);
11.单引号：
$b=1;
$a=’test$bdwqd\’wqdqw’;
var_dump($a);
echo $a;
在定义一些简单的字符串的时候使用单引号这种方法效率比较高，但是单引号中不能包含单引号。如果要单引号中包含另一个单引号需要（\）反斜杠来转义
注意：单引号字符串出现的变量不会被变量替代，即PHP不会解析单引号中的变量，因此在定义简单字符串时，使用单引号效率会更高，因此没有特别要求，应用单引号定义字符串。
12.双引号
$b=100;
$a=’te$bst……’;
$a=”te{$b}st……”;{}的作用是保证$b能够正常被PHP解析，否则$b
会变成变量$bst..,程序会因此报错，因为此变量并不存在
var_dump($a);
echo $a;
双引号中的变量名会被变量值所代替，即PHP会解析双引号之间的变量，同样双引号中不能放双引号，如果要放需要（\）来转义。
13.定界符
$b=200;
$a=<<<www
dwqdqwdwq{$b}dwqdqdqdqdwqdwqcwcfwefwefcscwecwefwdqwdqdefwefwefewfwe
www;
var_dump($a);
<<<之后提供一个标识符开始，然后是字符串的内容，最后同样的标识符表示结束，
注意：结束标识符之后必须接（;）不可以使用其他的任何字符，标识符的命名规则，字母数字下划线，并且不能以数字开始。定界符也会解析其中的变量。定界符的优点是，他可以包含单引号，和双引号。
14.变量类型相互转换
这些都是程序运行中不太可控的部分所以在编写代码时最好不要涉及!
强制转换:
$a=’上海’;
$b=(int)$a;
var_dump($a);
var_dump($b);
注意;强制转换并不会改变变量本身。
