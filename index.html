第四章 函数（一）自定义函数
函数的概念：
从功能上讲函数你可以把她看作是现实生活中的一个人或者一个公司里面的一个部门，可以接受我们的指令为我们完成某项固定的工作。
使用函数的优点：
、提高程序的重要性
、提高软件的可维护性
、提高软件的开发效率
、等等……
一、自定义函数
1.函数的定义：
function 函数名（[参数1，参数2，…参数n]）{
函数体；//任何有效的PHP代码都可以作为函数体使用，return表达式；//可以从函数中返回一个值，也可以不反回
}
注意：[]中的内容可以省略，函数名只能以字母、数字、下划线命名，且只能以字母、下划线开头。
2.函数的调用：
不管是自己定义的函数还是系统提供的函数，如果函数不被调用就不会执行。
调用：函数被调用后开始执行函数体中的代码，执行的过程是相对独立的，默认没有联系，互不干扰。执行完毕返回调用的位置继续向下执行，
通过定义好的 函数名（）来调用定义好的函数
注意：在定义好的函数之前也可以通过代码调用已经定义好的函数，这是PHP的一个特性。
3.函数的参数：
形式参数由零个、一个或者多个变量组成
实际参数由零个、一个或者多个参数组成，每个参数都是一个表达式，用逗号分隔
注意：形式参数定义几个，实际参数的个数需要与之对应，实际参数可多写，但多写的实际参数并不会上传到形式参数，也不会影响程序运行的最终结果，但不能少写，少写时会报错。
4.函数的返回值：
函数的返回值是将函数执行后的结果返回给调用者
默认如果不写返回值（return）则函数的返回值为null，默认返回null
return有两个作用，1是返回函数的值。 2是结束该函数的运行
第四章 函数（二）变量范围

一、PHP变量的范围
1.局部变量
局部变量也称为内部变量。局部变量是在函数内定义的，其作用域仅限于函数内部，离开该函数后在使用这种变量就是非法错误的！
示例：
function test(){
$a=1;//局部变量$a，尽在这个函数内部有效
}
echo $a;
2.全局变量
全局变量也称为外部变量，在函数的外部定义的，其作用域为从变量定义处开始，到本程序文件的末尾
在PHP函数中无法直接使用全局变量，如需使用或必须使用global声明变量
示例：
$i=10;//全局变量（外部变量）
define(‘MY_NAME’,’上海’);
function test(){
echo MY_NAME;
global $i;//声明$i为全局变量,然后再这个函数内部就可以使用$i
echo $i;
$a=1;
}
test();
3.静态变量

静态变量的是指在函数内部定义变量的时候使用static关键字来定义变量
特点：函数执行完毕之后变量不会立即消失，当再次调用函数时，静态变量保存的值依然存在，并且仅在第一次执行函数的时候会初始化值。
示例：
function test(){
static $a=10;//静态变量$a,初始化静态变量,尽在第一次调用的时候执行
echo ++$a;//当第二次执行这个函数的时候$a这个变量依然存在并且里面的也值依然存在！
}
test();

test();
echo ‘<换行br />’;
test();
echo ‘<换行br />’;
test();
echo ‘<换行br />’;
test();
echo ‘<换行br />’;
test();
echo ‘<换行br />’;
test();
第四章 函数（三）参数传递可变函数
一、参数的传递：
1.按值传递参数
默认的，我们在传递参数时默认使用的就是这个方式。
对形式参数的操作，根本不是影响到实际参数(变量)的值，两者相当于没有联系!
2.按引用传递参数
按引用传递参数，相当于形式参数和实际参数指的是同一个人，只是名字不一样，对于形式参数的操作会直接影响到实际参数（变量）;
示例：

}

可以给形式参数设置默认值，设置方法很简单直接赋值即可！
给形式参数加默认值的时候是从右往左加的，右边的必须有，才能够给左边的加！

二、可变函数（变量函数）
概念：如果一个变量名后有括号，PHP将寻找与变量的值同名的函数，并且尝试执行，这样就可以将函数的名称赋给变量，在程序中使用变量名并在后面加上圆括号即可执行对应的函数。
示例：
function test(){
echo’ 我是test函数’；
}
$a=’test’;//把函数名字加上引号，赋值给变量即可
$a();/test();
直接把函数赋值给变量
可变函数不能用于例如 echo，print，unset()，isset()，empty()，include，require 以及类似的语言结构。需要使用自己的包装函数来将这些结构用作可变函数。

第三章流程控制
流程控制：对PHP程序的执行过程进行控制，PHP有哪些手段对程序执行过程进行控制。
一、顺序执行
顺序执行非常简单，就是从上往下执行即可。对执行过程没有控制
二、分支执行
顺序执行虽然能解决计算，输出等问题，但是执行太过死板，无法完全满足我们的实际开发需要！所以我们需要引入分支执行，分支执行可以根据条件是否满足来选择执行某些代码，PHP的分支执行主要通过两种语句（if， swich）来实现。
1.if语句：我们if语句的表达式所控制的代码、最好用花括号括起来，哪怕就是一条一句也是用花括号。

概述：运算符是对一个或者多个操作数（变量或者数值）执行某种运算的符号，也称操作符
2.示例：! true !是运算符 true是操作数
$a + $b + 是运算符 $a，$b是操作数
true ? 1:0 ? 和 : 是运算符 true，1，0是操作数
根据操作数的个数分为，一元运算符、二元运算符、三元运算符
一些场合下在操作符的两边加上空格增加美观以及代码的可读性
3.算数运算符

示例：

echo $c;

6) ++ 自增
前++
当在一个语句里面需要使用到被自增的变量的值的时候，前++会先执行自增,然后再返回自增之后的值！
$a=10;

常量是用于临时（只有在我们程序运行过程中才存在）存储值（数据）的容器。
2.定义和使用

define（’常量名称’,常量值）或者 define（”常量名称”,常量值）使用常量名称时需要去掉（’’）或者（””）,当常量的值为字符串时，字符串中的内容需要用（’字符串’或者”字符串”）的形式，否则会报错。
常量的命名与变量类似，也遵循PHP标识符的名称规则，按照惯例常量标识符总是大写的
注意：常量的命名和变量一样，只能包含字母数字下划线，并且必须以字母或者下划线开头，按照惯例常量标识符总是大写的，自己定义的常量是严格区分大小写的
define（’MY_CITY’,’上海’）
defined（）函数来检查是否定义了某个常量

1）常量前没有美元符号（$）

5）常量的值只能是bool，int，float，string类型

2.变量的声明：
PHP的特性之一就是他不要求在使用变量前声明变量，当第一次给一个变量赋值时，你才创建了变量，变量用于储值比如数字、文本字符串或者数组。一旦设置了某个变量，我们就可以在脚本中重复使用它。PHP中变量必须使用一个美元符号$后面跟着变量名来表示，使用赋值操作符（=）给一个变量赋值。
3.声明：就是事先告知，通知！
注意：PHP程序中，使用变量前不需要声明，而其他语言是需要声明的，比如Java、c语言

unset（$变量的名称）函数释放指定变量
$and1=1;

6.变量的命名：
1）变量名是严格区分大小写的。
$name $name=1
$NAME $NAME=1
$Name $Name=1
2)变量名由字母或者下划线开头，后面可以连接任意数量的字母，数字，或者下划线。
注意：变量名的标识符一定不要以数字开头，中间不可以使用空白字符，不能使用点分开。
总结：变量名只能包含字母、数字、下划线，并且必须以字母或者下划线开头。
3）PHP中有一些标识符是系统定义的，也称为关键字，是PHP语言的组成部分，因此不能使用他们中的任何一个作为常量函数名或类名。但是和其他语言不同的是可以在PHP中作为变量名称使用，不过这样容易混淆，所以建议还是不要使用PHP的关键字作为变量名称。

Port指定一个端口号，可以省略默认为80
Abs_path被请求资源的位置
[]里面的内容可有可无，可以省略。
Dns服务器：
将域名和IP地址联系到一起，通过域名找到对应的IP地址，来实现访问
第一章PHP运行环境安装
1.需要安装的软件
1）web服务器
服务器名称：Apache服务器
软件安装地址：www.apache.org
2）PHP应用服务器
作用：解释执行编写的PHP程序
3）数据库管理系统/数据库服务器
服务器名称：MySQL
提示：Apache+PHP+MySQL是PHP比较流行的生存环境
2.wamp集成环境的安装
1）软件名称：wampserver
2）安装地址：www.wampserver.com
3）此集成环境包含Apache，PHP，MySQL
3.检验开发环境
通过浏览器访问127.0.0.1/localhost来检验环境安装是否正确
4.站点根目录
网站的默认存储位置就是站点根目录
5.根目录
1）根目录的默认位置为 安装盘符:\wamp\www
2）网络位置为http://localhost:80/(“:80”可省略，单独的一个“/”表示访问资源的根目录)
6.默认访问位置
访问某个文件夹会默认访问这个文件夹下的index.php/index.html文件，当文件夹下同时包含php和HTML文件时，wamp集成环境会优先输出PHP文件。
注意：网站的代码，图片以及相关文件夹应该放在站点根目录下，放到其他位置是无效的。

7.命名格式
PHP文件的命名格式为***.php
第一章PHP基础增强篇
1.编辑器安装
1）编辑器名称：Zendstudio
2）可选择汉化，但不建议汉化，汉化可能会影响部分功能的使用
3）安装时需要插入破解插件，然后输入特定的激活码完成激活使用
4）默认安装到C盘时可通过剪切，粘贴来完成位置的变换
2.编辑器的汉化
1）把需要的汉化包解压
2）打开zendstudio找到Help
3）左键单击Install New Software
4）点开add…左键单击Name后的local…选择汉化包的文件位置（juno），最后点击OK
5）在弹出的窗口中勾选Babel Language Packs in Chinese（Simplified），点击Next
6）等待Zendstudio重启后完成汉化
3.编辑器的配置
1）工作区的设置（设置代码存放的位置）
左键单击 file/文件，找到Switch Workspace/交换工作空间，点击Other…/其他，来完成位置的替换
2）红色的波浪线提示代码错误
3）字体的设置
左键单击Windows/窗口，点击Preferences/首选项，展开General/常规选项，找到Appearance/外观，点击Colors and Fonts找到View and Editor Folders/视图和编辑器字体，进行设置。
4）右击Close Project/关闭项目来关闭项目，此时项目无法编辑，再次右击单击Open Project打开项目来编辑
